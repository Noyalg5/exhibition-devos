## About

This is a concept demonstrator built for a Knowledge Transfer Partnership (KTP) Associate application with [Barker Langham](https://barkerlangham.com), a global cultural consultancy specialising in exhibition design and museum development. It uses a fictional touring exhibition — *ECHO: Memory in the Age of Machines* (London → Dubai → Riyadh) — as the case study to show how traditional linear production can be redesigned around digital collaboration, media-first thinking, and regenerative practice.

## What it shows

| Module | Description |
|--------|-------------|
| **Overview** | Tour route, headline outcomes, and the core thesis |
| **Process Mapper** | Side-by-side comparison of the conventional linear model vs. the proposed media-centric model |
| **Costing Model** | Live parametric budget tool — adjust area, reuse split, venues, freight and contingency to see costs and margins update in real time |
| **Sustainability Dashboard** | Toggle between build-new and reuse-modular modes to compare carbon footprint, waste diversion, water use, LCA score, and UN SDG alignment |
| **Collaboration** | Global team structure, city clocks, and a RACI responsibility matrix across the London, Dubai and Riyadh offices |

## Tech

Single self-contained `index.html` — vanilla JavaScript, inline CSS, hand-written inline SVG charts, zero dependencies. No build step. Runs offline from disk.

## Note on data

Figures are illustrative, modelled to show direction and scale rather than to represent any actual project.
